📦 BSscaner - Installers
==========================

✅ ФАЙЛОВОЕ ХРАНИЛИЩЕ ДАННЫХ

СОДЕРЖАНИЕ:
-----------

1. BSscaner-Apple-Silicon.dmg (90 MB)
   📱 Для Mac с чипом Apple Silicon (M1/M2/M3/M1 Pro/M1 Max/M1 Ultra)

2. BSscaner-Intel.dmg (187 MB)
   💻 Для Mac с процессором Intel (Core i5/i7/i9/Xeon)

УСТАНОВКА:
----------
1. Выберите подходящую версию для вашего Mac
2. Дважды кликните на файл .dmg
3. Перетащите BSscaner в папку Applications
4. Запустите из Applications

ПЕРВЫЙ ЗАПУСК:
-------------
⚠️ Начинается с чистого состояния
   - API ключи НЕ вшиты
   - Сессии НЕ сохранены

1. Откройте BSscaner
2. Настройки → Введите OpenRouter API ключ (https://openrouter.ai)
3. Аккаунт → Введите Bluesky handle и пароль
4. Готово!

ХРАНЕНИЕ ДАННЫХ:
---------------
📁 Все данные сохраняются локально:
   ~/Library/Application Support/BSscaner/bsscaner_data/
   
   ✅ Данные сохраняются при обновлении
   ✅ Изолированное хранилище для каждого пользователя

ФУНКЦИИ:
--------
✅ Генерация постов через ИИ
✅ Анализ трендов Bluesky
✅ Аналитика постов
✅ Отложенная публикация
✅ Нейрокомментирование
✅ Автоопределение языка (10 языков)

ТРЕБОВАНИЯ:
-----------
- macOS 10.13+
- Интернет
- Аккаунт OpenRouter (https://openrouter.ai)
- Аккаунт Bluesky
