BSscaner — Bluesky Social Scanner & Post Generator

Это мой pet-project — десктопное приложение для автоматизации работы с Bluesky.
Я делал его как инструмент для себя, чтобы упростить генерацию постов, анализ ленты и автокомментирование через ИИ.

Приложение позволяет:
	•	генерировать посты
	•	анализировать тренды
	•	автоматически комментировать популярные записи
	•	планировать публикации

⸻

Возможности
	•	Генерация постов через LLM (OpenRouter API)
	•	Анализ трендов из ленты Bluesky
	•	Аналитика постов (лайки, комментарии, репосты)
	•	Отложенная публикация
	•	Нейрокомментирование популярных постов
	•	Автоматическое определение языка
	•	Поддержка 10 языков

⸻

Технологии

Проект написан с использованием:
	•	Electron 28
	•	ATProto (Bluesky Protocol)
	•	OpenRouter API
	•	Google Material Icons

⸻

Требования
	•	Node.js 18+
	•	npm или yarn
	•	macOS 10.13+ (для сборки под macOS)

⸻

Установка

1. Клонирование репозитория

git clone <repository-url>
cd bsscaner

2. Установка зависимостей

npm install

3. Запуск в режиме разработки

npm run dev


⸻

Сборка

Сборка под macOS Intel (x64)

npm run build:mac-intel

Готовый .dmg файл будет в папке dist/.

Сборка для всех платформ

npm run build

Настройка

OpenRouter API
	1.	Открыть раздел Настройки
	2.	Вставить свой API-ключ
	3.	Выбрать модель и язык
	4.	Сохранить настройки

Подключение Bluesky
	1.	Перейти в раздел Аккаунт
	2.	Ввести handle и пароль
	3.	Нажать “Войти”

⸻

Использование

Генерация постов
	1.	Открыть раздел Посты
	2.	Ввести тему
	3.	Выбрать стиль:
	•	Вирусный
	•	Юмористический
	•	Философский
	•	Саркастичный
	4.	Выбрать язык
	5.	Нажать “Генерировать”
	6.	Опубликовать сразу или отложить

⸻

Нейрокомментирование
	1.	Открыть раздел Нейрокомментинг
	2.	Выбрать тип постов (свои или из ленты)
	3.	Настроить фильтры
	4.	Нажать “Начать”

Приложение автоматически находит популярные посты и публикует комментарии.

⸻

Отложенные публикации
	1.	Сгенерировать пост
	2.	Нажать “Отложить”
	3.	Выбрать дату и время

Пост будет опубликован автоматически.
