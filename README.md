# BSscaner - Bluesky Social Scanner & Post Generator

Desktop приложение для автоматизации работы с Bluesky: генерация постов, анализ трендов, нейрокомментирование и планирование публикаций.

## 🚀 Возможности

- **Генерация постов** с помощью ИИ (LLM)
- **Анализ трендов** с Bluesky ленты
- **Аналитика** постов (лайки, комментарии, репосты)
- **Отложенная публикация** с системными уведомлениями
- **Нейрокомментирование** популярных постов
- **Автоматическое определение языка** для комментариев
- **Поддержка 10 языков** контента

## 📋 Требования

- Node.js 18+ 
- npm или yarn
- macOS 10.13+ (для сборки macOS)

## 🛠️ Установка и сборка

### Разработка

```bash
# Клонировать репозиторий
git clone <repository-url>
cd bsscaner

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

### Сборка для macOS Intel (x64)

```bash
# Очистить данные и собрать DMG
npm run build:mac-intel

# Готовый установщик будет в папке dist/
```

### Сборка для всех платформ

```bash
npm run build
```

## 🔧 Конфигурация

### Настройка API ключа

1. Откройте приложение
2. Перейдите в раздел **Настройки**
3. Введите ваш **OpenRouter API ключ** (получите на [openrouter.ai](https://openrouter.ai))
4. Выберите модель и язык
5. Сохраните настройки

### Подключение Bluesky

1. Перейдите в раздел **Аккаунт**
2. Введите ваш handle и пароль Bluesky
3. Нажмите **Войти**

## 🎯 Использование

### Генерация постов

1. Раздел **Посты** → Введите тему
2. Выберите стиль (Вирусный, Юмористический, Философский, Саркастичный)
3. Выберите язык
4. Нажмите **Генерировать пост**
5. Опубликуйте или отложите публикацию

### Нейрокомментирование

1. Раздел **Нейрокомментинг**
2. Выберите тип постов (Свои или в ленте)
3. Настройте фильтры (минимум лайков, стиль комментария)
4. Нажмите **Начать комментирование**
5. Приложение автоматически находит популярные посты и комментирует их

### Отложенные публикации

1. Сгенерируйте пост
2. Нажмите **Отложить**
3. Выберите дату и время
4. Пост будет опубликован автоматически в выбранное время

## ⚠️ Внимание

- **Никогда не коммитьте файлы с токенами**: .gitignore настроен для защиты
- **Перед сборкой**: prebuild.js автоматически удаляет пользовательские данные
- **Безопасность**: пароли и токены хранятся локально в зашифрованном виде
